# rustvent

Advent of Code solutions in Rust

## Usage

```
rustvent 0.1.0
Advent of Rust.

USAGE:
    rustvent [OPTIONS] --day <day>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -d, --day <day>
    -i, --input <input>
    -p, --part <part>       [default: both]
    -y, --year <year>       [default: 2019]
```
